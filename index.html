Cygwin Dot Files

To use this in any cygwin install, execute this in the home directory:

$ curl http://caspian311.github.com/cygwin_dot_files/cygwin_dot_files.tgz | tar zx

This will download and un-package all the files that will make your Cygwin install work like a regular Linux distro.
